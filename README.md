# E-Commerce Frontend

A **static HTML/CSS/JS e-commerce storefront** — a multi-page shopping website with product listings, cart, user login/registration, and a deals section.

## Live Demo

🌐 [ahmedelneziliiy.github.io/Front-End-Project](https://ahmedelneziliiy.github.io/Front-End-Project/)

## Tech Stack

- **HTML5** / **CSS3** — modular stylesheets per section
- **Vanilla JavaScript**

## Pages

- `index.html` — Main storefront with hero, products, and deals
- `cart.html` — Shopping cart view
- Products loaded from `assets/products.json`

## Getting Started

Open `index.html` in a browser — no build step required.
