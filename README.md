# E-Commerce Frontend

A **static HTML/CSS/JS e-commerce storefront** — a multi-page shopping website with product listings, cart, user login/registration, and a deals section.

## What it does

A frontend-only e-commerce site built with vanilla HTML, CSS, and JavaScript. Includes a product grid loaded from a local JSON file, a shopping cart page, deals section, and user authentication pages.

## Live Demo

🌐 [ahmedelneziliiy.github.io/Front-End-Project](https://ahmedelneziliiy.github.io/Front-End-Project/)

## Tech Stack

- **HTML5**
- **CSS3** — modular stylesheets per section
- **Vanilla JavaScript**

## Pages

| Page | Description |
|---|---|
| `index.html` | Main storefront with hero, products, and deals |
| `cart.html` | Shopping cart view |
| Product listing with JSON data (`assets/products.json`) |

## Getting Started

Open `index.html` in a browser — no build step required.
